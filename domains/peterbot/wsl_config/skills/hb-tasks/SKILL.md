# Hadley Bricks Tasks

## Purpose

Shows today's workflow tasks generated by the Hadley Bricks system. These are automated task suggestions based on business rules (orders to ship, items to list, pricing to review, etc.). Conversational only.

## Pre-fetched Data

Data from the Hadley Bricks API:

- `data.tasks`: Array of today's tasks
  - `id`: Task ID (for completion)
  - `type`: Task type (ship, list, reprice, photograph, etc.)
  - `priority`: high, medium, low
  - `title`: Task title
  - `description`: What needs to be done
  - `item`: Related item/order if applicable
  - `due_time`: When it should be done by
  - `estimated_minutes`: How long it might take
  - `created_at`: When task was generated
- `data.completed_today`: Tasks completed today
- `data.total_estimate`: Total estimated time for all tasks
- `data.error`: Error message if fetch failed

## Rules

- Group by priority (high first)
- Show emoji indicators by priority
- Include actionable details (order IDs, set names)
- Show estimated time total
- Note what's been completed
- Keep items concise

## Task Types

- **ship**: Order needs dispatching
- **list**: Item needs listing on platforms
- **reprice**: Price adjustment recommended
- **photograph**: Item needs photos
- **review**: Review/audit task
- **pickup**: Collection to arrange
- **restock**: Low stock alert
